# QuoteForge Product Vision

QuoteForge exists to help tradespeople create professional proposals in minutes.

The software should remove paperwork, not create more of it.

Tradespeople should be able to write naturally.

AI should organise their notes into a professional proposal.

The customer should receive a proposal that creates confidence and trust.

Every feature should answer one question:

Does this help the tradesperson save time or win more work?

If not, it is probably not a priority.

---

## Principles

Simple beats clever.

Professional beats flashy.

Fast beats feature-heavy.

Consistency beats novelty.

The software should always feel calm, reliable and easy to use.

---

## Customer Experience

The customer should feel like they are dealing with a premium business.

The proposal is often the first impression.

Every proposal should increase trust.

---

## User Experience

The tradesperson should never need to think about the software.

They should simply describe the work.

QuoteForge should do the organisation.

The workflow should feel effortless.

---

## Long-Term Vision

QuoteForge should become the operating system for small trade businesses.

Not by adding hundreds of features.

But by making the essential features exceptional.

## Decision Filter

Before building any new feature, ask:

1. Does this save the tradesperson time?

2. Does this help them win more work?

3. Does this improve the customer's experience?

4. Can it be understood without training?

5. Would most tradespeople use it every week?

If the answer to most of these questions is "No", it should not be a priority.

## Invisible Quality

QuoteForge should quietly improve the user's work without requiring extra effort.

Where it is safe to do so, the application should automatically:

- Format names into title case.
- Remove leading and trailing spaces.
- Remove duplicate spaces.
- Normalise phone numbers.
- Store email addresses in lowercase.
- Correct simple formatting mistakes.
- Present information consistently throughout the application.

The tradesperson should never need to think about formatting.

QuoteForge should handle presentation automatically so the user can focus on the job.
