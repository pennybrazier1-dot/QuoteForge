# Mobile Home Reference

Visual reference:

`docs/design/mobile-home-reference.png`

This document defines the official mobile Home screen for the product.

This is the reference design for every future Home screen.

Desktop layouts should expand from this design.

---

# Philosophy

The Home screen is NOT a dashboard.

The Home screen answers one question:

> What do I need to do today?

A tradesperson should understand their day within five seconds.

No charts.

No analytics.

No KPIs.

No unnecessary information.

Every card should lead directly to an action.

---

# Target Device

Primary design width:

375px

Support:

390px

393px

402px

430px

(Modern iPhones and equivalent Android devices.)

---

# Header

Show:

Good morning, {First Name}

Today's date

Notification bell

Greeting changes automatically:

Morning

Afternoon

Evening

---

# Primary Question

Large heading:

"What do you need to get done today?"

This should always remain visible near the top.

---

# Sections

## Today's Jobs

Shows today's scheduled work.

Each card includes:

Customer

Job title

Time

Address

Status

Chevron

Tap opens Job.

---

## Jobs Needing Attention

Orange cards.

Examples:

Waiting for customer

Missing materials

Missing measurements

Date not confirmed

Tap opens the job.

---

## New Quote Requests

Draft proposals requiring completion.

Tap opens Proposal.

---

## Quotes Ready to Send

Ready-to-send proposals.

Primary action:

Send by Email.

---

# Cards

Cards should feel:

Friendly

Simple

Spacious

Large touch targets.

Rounded corners.

Subtle shadow.

No dense tables.

No tiny text.

One card = one action.

---

# Bottom Navigation

Five items.

Home

Customers

New

Calendar

More

New is centred.

New uses the brand orange.

Navigation is fixed.

Supports iPhone safe area.

---

# Colour

Support both Light Mode and Dark Mode.

Use system appearance automatically.

Light Mode:

White background

Dark text

Dark Mode:

Charcoal background

White text

Accent colours remain:

Green

Orange

Blue

Purple

---

# Design Principles

Phone first.

One-handed use.

Large touch targets.

Minimal typing.

Clear hierarchy.

Fast completion.

Calm.

Obvious.

Every screen should make the user feel more organised than they did ten seconds earlier.

# Product Mission

Finish work.

Not paperwork.
