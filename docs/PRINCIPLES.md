# QuoteForge Design Principles

These principles guide every product decision.

---

## Help, Don't Hinder

Every screen should make life easier.

Nothing should create unnecessary work.

---

## AI Assists. Humans Decide.

AI is a helper.

It is never the product.

The tradesperson remains in control.

---

## Save Time

Every feature should reduce administration.

If it doesn't save time, question why it exists.

---

## Reduce Mental Load

Users should never wonder:

"What do I do next?"

The software should guide them.

---

## Stay Organised

Everything should have a place.

Customers.

Proposals.

Jobs.

Documents.

Notes.

Nothing gets lost.

---

## Never Type It Twice

If information already exists, reuse it.

Never ask users for the same information repeatedly.

---

## Professional By Default

Every proposal should look professional before the user edits anything.

---

## Simple Beats Clever

Choose the simpler solution whenever possible.

---

## Finish The Day

The goal of QuoteForge is simple.

A tradesperson should be able to finish their paperwork quickly, close the laptop, and enjoy the rest of their evening.

---

## Every Screen Has One Job

Landing page

→ Build trust.

Dashboard

→ Show today's work.

Proposal screen

→ Create great proposals.

Customer page

→ Keep organised.

Settings

→ Configure once.
